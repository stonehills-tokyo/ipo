# IPO銘柄を投資するために必要な情報の整理方法

IPO（新規株式公開）銘柄への投資判断を行うために、収集すべき情報と整理の流れをまとめます。目的は「公開前後で比較できる形で、過去データと未来予測を区別しながら判断材料を揃える」ことです。

## 1. 基本情報（会社と上場条件の概要）
- **会社概要**: 事業内容、主要顧客、競合、収益モデル
- **上場市場**: プライム／スタンダード／グロースなど（市場の特徴も整理）
- **上場スケジュール**: ブックビル期間、上場日、ロックアップ解除日
- **主幹事証券**: 引受比率・過去の実績も参考にする
- **公開株数・売出比率**: 公募・売出の内訳、希薄化の度合い

## 2. 需給・需給イベント（需給バランスの把握）
- **公開規模（吸収金額）**: 時価総額と合わせて需給を判断
- **オファリングレシオ**: 既存株主の売出割合（高いほど需給悪化の可能性）
- **ロックアップ条件**: 解除価格・期間、VCの有無
- **同時期のIPO数**: 資金分散による需給悪化の可能性

## 3. バリュエーション（価格の妥当性）
- **想定価格・仮条件・公開価格**: 価格決定プロセスを時系列で整理
- **PER / PSR / EV/EBITDA**: 同業他社や過去IPOと比較
- **成長率とのバランス**: 売上成長率・利益成長率との整合性
- **ストーリー性**: 成長シナリオが現実的か

## 4. 業績と財務（過去実績と安定性）
- **売上・利益推移**: 直近3〜5年の成長トレンド
- **利益率**: 粗利率・営業利益率・EBITDAマージン
- **財務健全性**: 自己資本比率、キャッシュフロー
- **一時要因**: 補助金・特需・会計上の特殊要因の有無

## 5. リスク要因（下振れ要素の洗い出し）
- **業界リスク**: 規制、競争激化、技術変化
- **業績変動要因**: 主要顧客依存、人材依存、特定技術依存
- **株式需給リスク**: 大株主の売却意向、VC保有比率
- **ガバナンス**: 役員構成、監査体制

## 6. 投資判断に使う整理フォーマット（テンプレ）
- **企業概要**: 事業・市場・競合
- **需給**: 公開規模、ロックアップ、売出比率
- **価格**: バリュエーション比較、妥当性
- **業績**: 成長性、利益率、財務
- **リスク**: 主要懸念点
- **投資方針**: 初値狙い／中長期保有の別

## 7. 情報ソース
- **有価証券届出書・目論見書**
- **証券会社のIPOレポート**
- **同業他社の決算資料**
- **上場市場の開示資料**

## 8. 整理のポイント
- **時系列で更新**（仮条件→公開価格→初値）
- **定量と定性を分けて記録**
- **需給とバリュエーションを分離して評価**

---

このREADMEは、IPO投資の意思決定に必要な情報の整理方法をまとめたものです。実際の投資判断は、個別銘柄の状況やリスク許容度に応じて調整してください。
