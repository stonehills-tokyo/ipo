### 分析ポリシー

 - chatgpt/会社名.md を作成していく
 - ipolist.mdの上から順番に作成 (作成済みは除外)
 - 業務内容、財務分析を記載
 - リンクをReadMe.meの "### 銘柄分析" に記載
 - 投資対象指定の指示があった場合、"### 投資対象銘柄 " にリンクを記載
