# stonehills-tokyo/ipo の ipolist.md 掲載銘柄に関する購入適格性スクリーニング研究レポート

## エグゼクティブサマリー

本レポートは、GitHubリポジトリ（stonehills-tokyo/ipo）の **ipolist.md** に掲載されたIPO関連銘柄（合計22社）を起点に、READMEおよび同リポジトリ内の分析メモで示唆される購入条件（投資基準）に照らして「購入対象として適格」な銘柄を抽出した。

結論として、READMEの「投資対象銘柄」欄に既に掲載されている **ノースサンド（446A）** は、（リポジトリ内で明示的に使われている）定量基準の主要項目を満たすと確認でき、購入対象として適格と判定した。加えて、同基準を厳密に適用した場合、**NSグループ（471A）** も主要条件を満たし、購入対象として適格と判定した。一方、他の多くの銘柄は「PER基準超過」「利益率10%未達」「直近3年の赤字（=必須条件未達）」「直近3年の増益（=必須条件未達）」「未上場かつ赤字継続」などにより不適格となった。

## 方法論

本レポートの作業手順は、以下の通り。

1. **ipolist.md** と **README.md** を確認し、銘柄候補・掲載ルール・README側の運用方針（「投資対象銘柄」への掲載条件）を抽出。
2. ipolist.md の一覧（会社名・コード・上場日・市場区分）を抽出して表化。
3. 各銘柄の基本情報（業種、時価総額、上場市場、上場日、ティッカー）と主要指標（PER、ROE、営業利益率、当期純利益、過去損益推移など）を補完。
4. READMEおよびリポジトリ内分析ファイルに見られる「投資基準」（例：利益率10%以上、ROE15%以上、PER30倍以下、純利益5億円以上、過去3年赤字なし、過去3年増益傾向）を評価指標として採用し、「適格/不適格」を判定。

### README上の投資対象指定との整合

READMEの「投資対象銘柄」には、現時点でノースサンドのみが掲載されている。したがって、現行のREADME運用に厳密に従う最終リストは **ノースサンド単独**となる。一方で、本レポートの基準適合という観点では **NSグループも適格**であり、投資対象候補としてREADMEへの追加検討余地がある。
